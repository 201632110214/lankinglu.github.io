Indent the first line of the paragraph on the Web page, which is the most commonly used text formatting effect. CSS provides the text-indent property, which facilitates text indentation. By using the text-indent property, the first line of all elements can be indented into a given length, or even the length can be negative. The most common use of this property is to indent the first line of a paragraph, and the following rules will indent the first line of all paragraphs into 5 em: P {text text-indent: 5 em; } Note: in general, text-indent can be applied to all block-level elements, but it is not possible to apply this attribute to the inline elements, and the text-indent attribute cannot be applied to the replacement elements such as images. However, if there is an image in the first line of a block-level element (such as a paragraph), it moves with the rest of the line. Tip: if you want to "indent" the first line of an inline element, you can create this effect with the left inner margin or outer space.
