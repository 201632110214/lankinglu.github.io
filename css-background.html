CSS allows you to apply solid colors as backgrounds, and also allows you to create fairly complex effects using background images. The ability of CSS to do this is far from HTML. You can use the background-color property to set the background color for the element. This property accepts any valid color value. This rule sets the element's background to grey: P {background - color: gray; } If you want the background color to be very few extensions from the text in the element, just add some inner margins: P {background - color: gray; Padding: 20 px; } If you want to see the effect of this example, try it yourself! You can set the background color for all the elements, including the body, all the way up to em and a. Background -color cannot be inherited, and its default value is transparent. Transparent means "transparent". That is, if an element does not specify a background color, the background is transparent so that the background of its ancestor elements can be seen.
